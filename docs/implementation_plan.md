# 주식 테마 및 수익성 분석기 구현 계획서

이 계획서는 주식 시장의 주요 테마를 식별하고, 거래량과 수익성을 기준으로 필터링하여 사용자에게 알림을 보내는 개인 프로젝트의 아키텍처를 설명합니다.

## 사용자 검토 필요 사항

> [!IMPORTANT]
> **API 키 구성**: 이 프로젝트의 모든 기능을 활용하려면 다음 외부 API 키가 필요합니다:
> 1. **DART API Key**: 재무제표를 통해 실제 수익성(매출액, 영업이익)을 확인하는 데 사용됩니다.
> 2. **카카오 REST API 키**: [카카오 개발자 센터](https://developers.kakao.com/)에서 앱을 생성하고 '나에게 메시지 보내기' 권한을 설정해야 합니다.
>
> [!NOTE]
> 네이버 증권은 테마 정리가 잘 되어 있어 '테마 수집'에는 가장 대중적입니다. 더 공식적인 데이터가 필요하다면 **한국거래소(KRX)**나 **증권사 API(한국투자증권 등)**를 사용할 수 있지만, 설정이 더 복잡해집니다.

## 제안하는 아키텍처

### 1. 기술 스택
- **언어**: Python 3.10+
- **데이터 수집**: 
  - `requests` + `BeautifulSoup`: 네이버 증권 또는 다음 금융의 테마 데이터 크롤링용.
  - `pykrx` / `FinanceDataReader`: 한국거래소(KRX) 공식 데이터(가격, 거래량) 수집용.
  - `dart-fss`: 금융감독원 DART API를 통한 실제 수익성(영업이익) 검증용.
- **알림 및 시각화**: 
  - **카카오톡 메시지 API (나에게 보내기)**.
  - **GitHub Pages**: 수집된 데이터를 대시보드 형태로 시각화하여 웹페이지로 제공.
  - **GitHub Actions**: 매일 정해진 시간에 프로그램을 자동 실행하고 웹페이지를 업데이트하는 자동화 엔진.

### 2. 기능별 구성 요소

#### Component A: 테마 수집기 (Theme Collector)
- `https://finance.naver.com/sise/theme.nhn` 페이지를 크롤링합니다.
- 당일 상승률 또는 거래량이 높은 상위 10개 테마를 추출합니다.

#### Component B: 종목 필터링 (Volume Filter)
- 선정된 상위 10개 테마 내의 모든 구성 종목을 가져옵니다.
- 종목들을 **거래량(Volume)** 기준으로 정렬합니다.
- 각 테마 내에서 거래량이 활발한 주요 후보 종목들을 선별합니다.

#### Component C: 수익성 검증기 (Profitability Validator)
- `dart-fss`를 사용하여 최신 연결재무제표를 조회합니다.
- 다음과 같은 기업은 필터링하여 제외합니다:
  - 영업손실(적자)이 발생한 기업.
  - (선택 사항) 매출액이 감소 추세인 기업.
- 최종적으로 조건에 부함하는 "우량" 종목 10개를 확정합니다.

#### Component D: 알림 전송기 (Notifier)
- 결과를 깔끔한 메시지 형식으로 구성합니다.
- **텔레그램 봇**을 통해 사용자에게 메시지를 전송합니다.

## 구현 단계

### 1단계: 환경 설정
- 프로젝트 디렉토리 생성 (예: `stock-notifier`).
- 가상환경 설정 및 라이브러리 설치: `pandas`, `requests`, `pykrx`, `dart-fss`, `python-telegram-bot`.

### 2단계: 테마 및 거래량 로직 구현
- 네이버 증권 크롤러 구현.
- `pykrx`를 이용한 한국거래소(KRX) 데이터 연동.

### 3단계: 재무제표 분석 로직
- DART API 키 연동 및 `dart-fss` 통합.
- 영업이익 흑자 여부를 판별하는 필터링 로직 작성.

### 4단계: 대시보드 및 자동화 구축
- GitHub Pages용 대시보드 웹페이지(HTML/CSS) 디자인.
- GitHub Actions 워크플로우(`update.yml`) 설정.
- 카카오톡 인증 토큰을 GitHub Secrets에 안전하게 저장.
- 전체 프로세스(수집-검증-알림-배포) 통합 테스트.

## 데이터 소스 비교 및 추천

| 소스 | 특징 | 추천 용도 |
| :--- | :--- | :--- |
| **네이버/다음 금융** | 접근성 좋음, 테마 카테고리가 잘 분류됨 | **테마 랭킹 및 구성 종목** 수집 |
| **한국거래소 (KRX)** | 공신력 있는 공식 데이터 (pykrx 사용) | **정확한 시세 및 거래량** 확인 |
| **DART (금감원)** | 기업이 직접 공시한 원본 재무제표 | **영업이익, 매출액 등 수익성** 검증 |
| **증권사 API (한투 등)** | 실시간성 가장 높음, 주문 기능 가능 | 실시간 트레이딩 및 고급 분석 |
