# 미국 주식(해외 우량주) 턴어라운드 모듈 추가 계획

현재 국내 주식 전용으로 설계된 파이프라인에, **미국 S&P 500 주식**을 대상으로 하는 저평가 턴어라운드 검색 모듈을 추가합니다.

## 제안하는 구현 방식 (통합 파이프라인)

### 1. 데이터 소스 및 라이브러리 추가
- **yfinance (`pip install yfinance`)**: 야후 파이낸스 공식 API를 래핑한 라이브러리를 통해 티커 리스트, 가격, 재무 정보를 모두 무료로 빠르게 수집합니다.
- S&P 500 기업 리스트(위키피디아 기반)를 긁어와 이 500대 우량 기업을 대상으로 스캔을 진행합니다.

### 2. 백엔드 로직 (`main.py` 리팩토링)
새로운 함수 `find_us_turnaround_stocks()`를 추가합니다.
- **1차 필터 (기본 지표)**: 
  - S&P 500 종목들을 순회하며 `yfinance`의 `.info` 객체를 로드.
  - 시가총액(Market Cap) > $1B (약 1.3조 원 이상)
  - PER < 20, PBR < 3 (미국 시장 평균을 고려하여 국내보다 컷을 약간 상향)
- **2차 필터 (차트 및 재무)**:
  - 1차를 통과한 종목들의 최근 6개월 일봉 데이터(`history`)를 가져와 5/20/60/120일선 정배열 턴어라운드 여부 검사.
  - `.quarterly_financials`를 열람하여 'Operating Income(영업이익)' 우상향 또는 흑자 전환 여부 체크.
- **최종 저장**: 통과한 10개 종목을 기존과 동일한 형태의 JSON 포맷으로 생성하되, 테마명을 `미국우량주(턴어라운드)`로 지정하여 `data.json`에 합칩니다.

### 3. 프론트엔드 업데이트 (`index.html`, `script.js`)
- 대시보드 상단 탭에 **[미국 우량주]** 버튼을 추가합니다.
- 해당 버튼 클릭 시 `theme === '미국우량주(턴어라운드)'` 인 종목만 정렬되어 나타나도록 `script.js`의 `filterData` 로직을 업데이트합니다.
- 화폐 단위가 $ 와 ₩ 로 다르므로, 미국 주식일 경우 UI상에서 적절히 표시할 수 있다면 반영합니다 (PER, PBR은 동일).

## 확인이 필요한 항목 (User Review)
